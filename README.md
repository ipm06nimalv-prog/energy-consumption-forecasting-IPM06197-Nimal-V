# energy-consumption-forecasting-IPM06197-Nimal-V
Energy consumption forecasting using clustering and machine learning

## Methods Used
- Data preprocessing using Pandas
- K-Means clustering to group similar households
- Random Forest Regressor
- Gradient Boosting Regressor

## Evaluation Metrics
- RMSE
- MAE
- R² Score

## Results
Gradient Boosting achieved the best performance with R² = 0.986.

Clustering slightly improved prediction accuracy by grouping households with similar energy usage patterns.
